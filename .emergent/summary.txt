The Koala Mining application is a Bitcoin mining simulator with real financial integrations. It features a dashboard (simulated BTC, hash rate, miners), 4-tab navigation, free/ad miners, an in-app store, a referral system, and a profile section. Authentication is via Google/email. Key enhancements include a black marble and gold theme, Forgot Password flow, real BTC withdrawals (min 0.00001 BTC, 0.5% fee absorbed network fees), auto-activating purchased miners, and a loading progress bar. Real-time balance updates (1-second), accurate Total Earned/Total Cashed Out, and 14-decimal display for BTC amounts are crucial. Monetization involves real Bitcoin withdrawals (currently via Kraken API) and PayPal for miner purchases, with Facebook Ads integrated for app launch, ad miners, and withdrawal clicks (0-30 daily ad counter, non-rewarded for launch/withdrawal). User feedback has driven refinements in ad logic, dashboard layout, withdrawal process (now 0.5% fee to a specified address), and live BTC pricing. The app has recently been rebranded to Koala Mining with a new logo.
